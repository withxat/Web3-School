# 🎓 AI × Web3 School — Personal Learning Journal

> Personal learning journal and proof-of-work for [AI × Web3 School](https://aiweb3.school/).

## 📚 About

[AI x Web3 School](https://aiweb3.school/) is an open-source learning platform by [LXDAO](https://lxdao.io/) and [ETHPanda](https://www.ethpanda.org/), covering AI fundamentals, Web3 fundamentals, and the intersection of both.

- 📖 [Handbook (中文)](https://aiweb3.school/zh/handbook/)

- 📋 [WCB Learning Page](https://web3career.build/zh/programs/AI-Web3-School#tab=learning)

## 📅 Daily Workflow

1. Read Handbook chapter → take notes in `daily/YYYY-MM-DD.md`
2. Complete tasks → document in `tasks/`
3. Write check-in draft → submit on WCB platform
4. Log feedback → file in `handbook-feedback/`
