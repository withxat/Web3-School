# 📝 Handbook Feedback

本目录用于记录学习过程中对 [AI × Web3 School Handbook](https://aiweb3.school/zh/handbook/) 的反馈。

## 反馈类型

- **🐛 错误:** 错别字、代码错误、链接失效
- **❓ 不清楚:** 概念解释不够清晰，需要更多上下文
- **📅 过期:** 资料或工具版本已过时
- **💡 建议:** 结构优化、补充内容、新增章节

## 文件命名规范

```
YYYY-MM-DD-简短描述.md
```

例如：
- `2026-05-19-network-rpc-endpoint.md`
- `2026-05-20-wallet-sign-flow.md`

## 反馈模板

每条反馈应包含：

1. **Handbook 页面链接** — 你在哪里看到的问题
2. **问题描述** — 具体是什么问题
3. **建议改法** — 你认为应该怎么改
4. **来源** — 你的判断依据（文档、实验、社区讨论等）

## 反馈的价值

这些反馈会被开源沉淀，帮助 Handbook 持续改进。每条认真写的反馈都是 proof-of-work。
